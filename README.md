# 👁 EyeBody — 내 아이 근시 관리 플랫폼

> 근시 억제 치료를 받는 아이의 안축장·치료 순응도·생활습관을 장기 추적하는 디지털 근시 관리 서비스

**[🔗 Demo 바로보기](https://halinaCho.github.io/Eyebody)**

---

## 주요 기능 (MVP)

| 화면 | 기능 |
|------|------|
| 홈 | 오늘의 치료 현황 · 안축장 상태 · 다음 진료일 · 순응도 · 생활습관 요약 |
| 기록 | 검사기록(안축장/SER) · 치료기록 · 생활습관(야외활동/스마트폰) 입력 |
| 분석 | 안축장 추이 그래프 · SER 변화 · 월별 치료 순응도 · 성장률 분석 |
| 설정 | 자녀 프로필 · 보호자 관리 · 알림 설정 · 계정 |

## 안축장 성장률 기준

| 단계 | 기준 | 색상 |
|------|------|------|
| Stable | < 0.20 mm/년 | 🟢 |
| Watch | 0.20 ~ 0.35 mm/년 | 🟡 |
| Rapid | > 0.35 mm/년 | 🔴 |

## 기술 스택 (MVP 웹 데모)

- HTML5 / CSS3 / Vanilla JS
- Chart.js (그래프)
- GitHub Pages (호스팅)

## 개발 로드맵

- **Phase 1 MVP** — 기록 · 순응도 · 분석 (현재)
- **Phase 2** — Supabase 연동 · 인증(카카오/네이버/구글)
- **Phase 3** — OCR 검사결과 자동입력
- **Phase 4** — AI 근시 위험도 예측
- **Phase 5** — 병원 연동 · 의사 계정
